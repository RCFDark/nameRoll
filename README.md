# 名字抽选器

一个可爱风格的班级抽名工具，支持滚动抽取和大转盘模式。

## 功能特性

- 📊 支持导入 Excel (.xlsx) 名单
- 🎯 两种抽取模式：滚动抽取 / 大转盘
- 👥 支持抽取1人、2人或自定义（最多10人）
- 📜 抽取历史记录
- 🔄 去除重复抽取选项
- 🎉 烟花和彩带特效
- 💾 本地存储，刷新不丢失

## 使用方法

1. 直接在浏览器打开 `index.html`
2. 点击「导入Excel」按钮，选择名单文件
3. 选择抽取模式（滚动/转盘）
4. 设置抽取人数
5. 点击「开始抽取」

## 部署到 GitHub Pages

1. 创建新仓库，将代码推送上去
2. 进入仓库 Settings → Pages
3. Source 选择 "Deploy from a branch"
4. Branch 选择 "main"，目录填写 "/"
5. 保存后等待部署完成即可访问

## 名单格式

Excel 文件中，名字应在第一列（或其他任意非空列，系统会自动识别）。
